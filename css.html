Types of CSS

  1. Inline Css

    2. Inline Css is added to the elements directly using the style attributes

  2. Internal Css

    3. Internal Css is kept inside the head tags in {style} tag

  3. External Css

    4. External Css is kept separately inside a .css style sheet.it involes two steps

    • First, write the css in .CSS file
    • include that CSS file to Markup
